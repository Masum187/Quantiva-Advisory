# Admin Dashboard

Willkommen im **Quantiva Admin-Dashboard**. Diese Seite dokumentiert Struktur, Bulk-Aktionen, Validierung, Undo/Redo/History und den Workflow.

## Übersicht

- **Listenansicht & Galerie** (umschaltbar)
- **Suche/Filter**: Kategorie, Branche, Volltext
- **Bulk-Aktionen**: Kategorie/Branche/Tech setzen/entfernen, Clear
- **Validierung**: Live-Fehler je Case
- **Undo/Redo + History**: Batching (300 ms), Toast mit „Rückgängig"
- **Workflow**: Entwurf → Prüfung → Freigabe → Veröffentlichung

## Felder (CaseItem)

- `slug` *(Pflicht)*  
- `titleDe` / `titleEn`  
- `subtitleDe` / `subtitleEn`  
- `category` / `industry`  
- `heroImage` / `heroMedia`  
- `goalsDe` / `goalsEn` / `solutionDe` / `solutionEn`  
- `resultsDe` / `resultsEn`  
- `tech[]`  
- `quote` *(optional)*  
- **Workflow**: `status`, `owner`, `reviewers[]`, `publishedAt`

## Validierungsregeln (Kurz)

- `slug`: `[a-z0-9-]+`
- `titleDe` **oder** `titleEn` muss existieren
- `heroImage`: `/....(jpg|jpeg|png|webp)`
- `heroMedia`: `/....(mp4|webm)`
- `quote`: TextDe **oder** TextEn muss existieren

## Workflow-Aktionen

- **Zur Prüfung** (Admin, Editor) → `inReview`
- **Freigeben** (Admin, Reviewer, Publisher) → `approved`
- **Ablehnen** (Admin, Reviewer) → `rejected`
- **Veröffentlichen** (Admin, Publisher) → `published` (+`publishedAt`)
- **Unpublish** (Admin, Publisher) → `approved`
- **Entwurf** (Admin) → `draft`

## Tipps

- **Owner/Reviewer** im Drawer pflegen
- **History export/import** für Audits
- **Dark-Mode** via Topbar
